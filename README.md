# D{0-1}KP Problem Solver

D{0-1}背包问题求解器 - Java实现

## 项目简介

本项目是一个用于求解D{0-1}背包问题的Java应用程序。

**D{0-1}KP问题特点：**
- 数据集由一组**项集**组成
- 每个项集有**3个物品**可供背包装入选择
- 第三项价值 = 前两项价值之和
- 第三项重量 < 前两项重量之和
- 每个项集的3个物品中**至多选择1个**装入背包
- 目标：在不超过背包容积的条件下，使装入背包的物品价值之和最大

## 功能特性

1. **数据读取**: 正确读入实验数据文件的有效D{0-1}KP数据
2. **散点图绘制**: 绘制任意一组D{0-1}KP数据以重量为横轴、价值为纵轴的数据散点图
3. **排序功能**: 对一组D{0-1}KP数据按项集第三项的价值:重量比进行非递增排序
4. **最优求解**: 用户能够自主选择指定D{0-1}KP数据的求解算法并获得最优求解结果
5. **结果保存**: 任意一组D{0-1}KP数据的最优解、求解时间和解结果可保存为txt文件或导出EXCEL文件
6. **友好界面**: 提供图形用户界面(GUI)和命令行界面(CLI)两种操作方式

## 数据文件格式

```
IDKP1:
尺寸=3*100
容积=61500
价值=408,921,1329,11,998,1009,...
重量=508,1021,1321,111,1098,1196,...
```

### 格式说明

| 关键字 | 说明 | 示例 |
|--------|------|------|
| `尺寸` | 数据规模，格式为 `3*N`（N为项集数量） | `3*100` 表示100个项集，每个项集3个物品 |
| `容积` | 背包最大容量 | `61500` |
| `价值` | 所有物品的价值列表，逗号分隔 | `408,921,1329,...` |
| `重量` | 所有物品的重量列表，逗号分隔 | `508,1021,1321,...` |

### 数据约束

- 物品总数 = 项集数 × 3
- 第3个物品的价值 = 第1个 + 第2个
- 第3个物品的重量 < 第1个 + 第2个

详见 [DATA_FORMAT.md](DATA_FORMAT.md)

## 编译和运行

### Windows

```bash
# 编译
compile.bat

# 运行图形界面
run.bat

# 运行命令行模式
run.bat data.txt
```

### Linux/Mac

```bash
# 编译
./compile.sh

# 运行图形界面
./run.sh

# 运行命令行模式
./run.sh data.txt
```

## 使用说明

### 图形界面模式

1. 启动程序后，点击"浏览..."按钮选择数据文件
2. 从下拉列表中选择一个实例
3. 选择求解算法
4. 点击相应按钮执行操作

### 命令行模式

```bash
java -cp "lib/*:bin" dkp.Main data.txt
```

## 算法说明

| 算法 | 时间复杂度 | 特点 |
|------|-----------|------|
| 动态规划 | O(n × C) | 保证最优解 |
| 贪心算法 | O(n log n) | 速度快，近似解 |
| 回溯算法 | O(4^n) | 精确解，小规模 |
