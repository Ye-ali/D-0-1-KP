# D{0-1}KP Problem Solver - 项目总结

###  已实现的功能

1. **数据读取功能**
   - 正确读取实验数据文件的有效D{0-1}KP数据
   - 支持解析10组标准测试实例
   - 自动验证数据约束条件

2. **散点图绘制功能**
   - 绘制任意一组D{0-1}KP数据的散点图
   - 以重量为横轴、价值为纵轴
   - 使用JFreeChart库实现
   - 支持保存为图片

3. **排序功能**
   - 按项集第三项的价值:重量比进行非递增排序
   - 提供详细的排序结果输出

4. **求解功能**
   - 用户可自主选择求解算法
   - 实现三种求解算法：
     - 动态规划算法（最优解）
     - 贪心算法（快速近似解）
     - 回溯算法（小规模精确解）

5. **结果保存功能**
   - 支持保存为txt文件
   - 支持导出为Excel文件
   - 包含完整的求解信息

6. **用户界面**
   - 友好的图形用户界面（GUI）
   - 命令行界面（CLI）
   - 支持两种模式切换

###  使用说明

#### 图形界面模式

1. 点击"Browse..."选择数据文件
2. 从下拉列表选择实例
3. 选择求解算法
4. 点击按钮执行操作：
   - Show Scatter Plot - 显示散点图
   - Sort by Ratio - 按比率排序
   - Solve - 求解问题
   - Save as TXT/Excel - 保存结果

#### 命令行模式

```bash
java -cp "lib/*:bin" dkp.Main idkp1-10.txt
```

然后按提示操作：
1. 选择实例编号
2. 选择操作选项
3. 查看或保存结果

###  算法说明

| 算法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|-----------|-----------|------|
| 动态规划 | O(n×C) | O(n×C) | 最优解，中等规模 |
| 贪心 | O(n log n) | O(n) | 快速，近似解 |
| 回溯 | O(4^n) | O(n) | 精确解，小规模 |

所有测试实例均满足D{0-1}KP问题的约束条件：
- 第三项价值 = 前两项价值之和
- 第三项重量 < 前两项重量之和
- 每个项集至多选择一个物品
