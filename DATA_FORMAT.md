# D{0-1}KP 数据格式说明

## 概述

D{0-1}KP问题的数据文件采用键值对格式，使用中文关键字便于理解。

## 数据格式

```
实例名称:
尺寸=3*N
容积=C
价值=p1,p2,p3,p4,p5,p6,...
重量=w1,w2,w3,w4,w5,w6,...
```

## 关键字说明

| 关键字 | 英文别名 | 说明 | 必需 |
|--------|----------|------|------|
| `尺寸` | `DIMENSION`, `D`, `d` | 数据规模，格式为 `3*N` | 是 |
| `容积` | `CAPACITY`, `C`, `c` | 背包最大容量 | 是 |
| `价值` | `PROFIT`, `PROFITS`, `P`, `p` | 物品价值列表 | 是 |
| `重量` | `WEIGHT`, `WEIGHTS`, `W`, `w` | 物品重量列表 | 是 |

## 尺寸说明

`尺寸=3*N` 表示：
- `3`：每个项集包含3个物品
- `N`：共有N个项集
- 总物品数 = 3 × N

例如：`尺寸=3*100` 表示有100个项集，共300个物品。

## 数据约束

### 1. 数量约束

- 物品总数必须是 **3的倍数**
- 物品总数 = 项集数 × 3

### 2. 第三项价值约束

对于每个项集（3个物品为一组）：

```
价值[3] = 价值[1] + 价值[2]
```

验证示例：
```
项集0: 价值=408,921,1329
验证: 1329 = 408 + 921 ✓
```

### 3. 第三项重量约束

对于每个项集（3个物品为一组）：

```
重量[3] < 重量[1] + 重量[2]
```

验证示例：
```
项集0: 重量=508,1021,1321
验证: 1321 < 508 + 1021 = 1529 ✓
```

## 创建数据文件

### 方法1：手动编辑

使用文本编辑器创建 `.txt` 文件，按照上述格式填写数据。

### 方法2：程序生成

```java
// 生成D{0-1}KP数据
public void generateData(int setCount, int capacity) {
    List<Integer> profits = new ArrayList<>();
    List<Integer> weights = new ArrayList<>();
    
    for (int i = 0; i < setCount; i++) {
        // 随机生成前两个物品
        int p0 = (int)(Math.random() * 500) + 50;
        int p1 = (int)(Math.random() * 500) + 50;
        int w0 = (int)(Math.random() * 500) + 100;
        int w1 = (int)(Math.random() * 500) + 100;
        
        // 第三项：价值=前两项之和
        int p2 = p0 + p1;
        // 第三项：重量<前两项之和
        int w2 = (int)(Math.random() * (w0 + w1 - 50)) + 50;
        
        profits.add(p0); profits.add(p1); profits.add(p2);
        weights.add(w0); weights.add(w1); weights.add(w2);
    }
    
    // 输出
    System.out.println("尺寸=3*" + setCount);
    System.out.println("容积=" + capacity);
    System.out.println("价值=" + String.join(",", profits));
    System.out.println("重量=" + String.join(",", weights));
}
```

## 常见问题

### Q1: 尺寸格式错误？

正确格式：`3*100` 或 `300`
- `3*100`：3个物品/项集 × 100个项集
- `300`：直接指定总物品数（程序自动除以3）

### Q2: 价值和重量数量不匹配？

```
错误：价值数量(299)与重量数量(300)不匹配！
```

解决：确保两个列表的数值数量相同。

### Q3: 物品数量不是3的倍数？

```
错误：物品数量(299)必须是3的倍数！
```

解决：确保物品总数能被3整除。

### Q4: 容积未指定？

```
容积未指定，自动设置为：XXXXX
```

程序会自动计算为总重量的一半。

## 注释

数据文件支持注释行，以 `#` 或 `//` 开头：

```
# 这是注释
// 这也是注释
尺寸=3*100
容积=61500
...
```
